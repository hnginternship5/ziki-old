## Installation
 **This project works with PHP**
* `git clone https://github.com/hng5-blog-task/blog-task.git`
* cd into the folder or directory
#Step 1
* ```php -S localhost:8000```
* open browser ``http://localhost:8000``    
OR
#Step 2
* ``` use XAMPP or WAMPP```
